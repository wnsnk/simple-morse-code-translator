# Simple Morse Code Translator
## Description
This is a simple text-based Morse Code Translator I made for the assignment on Day 82 of the 100 Days of Code course. 
This program can translate text to Morse Code and Morse Code back to text.

![Morse Code Screenshot](/screenshots/morse%20code%20translator.png "Morse Code Translator")
## Installation
```bash
git clone https://github.com/wnsnk/simple-morse-code-translator.git
cd simple-morse-code-translator
python main.py
```
